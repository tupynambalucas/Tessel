# 🌐 Social Hub Roadmap (@tessel/www)

Focus on building the asynchronous social layer, unified identity, and marketplace integration.

## 🏁 Phase 1: Identity & Foundation

- [x] React 19 / Tailwind 4 project initialization.
- [ ] **Unified Auth (SSO):** Hub as the master identity provider for Web/Desktop instances.
- [ ] User profile customization (Bios, Avatars, Social links).
- [ ] Basic landing pages and product discovery.

## 📱 Phase 2: Asynchronous Social Layer

- [ ] **Timeline Implementation:** Global and following feeds for status updates.
- [ ] **Communities & Groups:** Dedicated spaces for niche social circles.
- [ ] **Private Messaging:** Real-time text-based chat (Socket.io).
- [ ] Media processing pipeline (Optimizing image/video uploads).

## 💰 Phase 3: Connected Experience & Economy

- [ ] **Media Bridge:** Social feed integration for in-game screenshots/captures.
- [ ] **Marketplace Core:** Shopping interface for 3D assets and customizations.
- [ ] **Inventory Hub:** Centralized management of acquired items equippable in 3D.
- [ ] Notification engine (Push and Web) for social and economic events.
