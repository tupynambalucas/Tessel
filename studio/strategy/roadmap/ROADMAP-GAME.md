# 🎮 Game Engine Roadmap (@tessel/game)

Focus on rendering performance, immersive visuals via WebGPU/TSL, and responsive client-side interactions.

## 🏁 Phase 1: Rendering & Movement
- [x] WebGPU Renderer initialization.
- [x] Basic TSL Material pipeline.
- [ ] Character controller with Rapier (Client-side prediction).
- [ ] Animation state machine integration (Idle/Walk/Run/Jump).
- [ ] Adaptive LOD (Level of Detail) system for environments.

## 🌈 Phase 2: Visual Fidelity & FX
- [ ] Advanced TSL Shaders (Water, Dissolve, Shields).
- [ ] Post-processing pipeline (Bloom, SSAO, Color Grading).
- [ ] Dynamic lighting and shadow optimization for WebGPU.
- [ ] Particle system (GPU-accelerated).

## 🗣️ Phase 3: Spatial Immersion
- [ ] Positional Audio listener integration.
- [ ] Emote system with synchronized animations.
- [ ] Interactive world objects (Doors, Buttons, Triggers).
